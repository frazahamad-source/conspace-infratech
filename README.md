# Conspace Infratech Website

This is the professional website for **Conspace Infratech**, a Real Estate, Interiors, and Construction Partnership Firm in Mangalore.

## Project Structure
- `index.html`: The main homepage.
- `css/style.css`: All styling and responsive design rules.
- `js/main.js`: Interactivity for slider and mobile menu.
- `assets/images/`: Folder for storing your images.

## How to Edit Content
1.  **Images**: Place your images in `assets/images/`.
2.  **Text**: Open `index.html` in a text editor (like Notepad or VS Code) and look for the text you want to change (e.g., project names, descriptions).
3.  **Team/Projects**: Look for the comments in `index.html` (e.g., `<!-- REPLACE ... -->`) to know where to update image paths.

## How to Publish for Free

### Option 1: Netlify Drop (Easiest)
1.  Go to [app.netlify.com/drop](https://app.netlify.com/drop).
2.  Drag and drop your entire `ConspaceInfratech` folder onto the page.
3.  Netlify will instantly publish your site and give you a unique URL (e.g., `random-name.netlify.app`).
4.  You can change the site name in "Site Settings" -> "Change site name".

### Option 2: GitHub Pages (Best for updates)
1.  Create a free account on [GitHub.com](https://github.com/).
2.  Create a new repository named `conspace-infratech`.
3.  Upload your files to this repository.
4.  Go to **Settings** -> **Pages**.
5.  Select `main` branch and save.
6.  Your site will be live at `https://yourusername.github.io/conspace-infratech`.

## Need Help?
If you have a custom domain (e.g., `conspaceinfratech.com`), you can connect it in the settings of either Netlify or GitHub Pages.
